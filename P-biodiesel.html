Modified Biodiesel Mini-Production Plant Using Used Cooking Oil

This was my degree capstone project from the Philippines. We were in a group of six, and I was assigned the task to 3D model and animate the project using Blender. We used glass vessels for easier inspection of the process. Unlike petrodiesel, biodiesel doesn't produce dark smoke exhaust. The project was a success, we used our biodiesel to run an engine an no dark exhaust was seen.
